Проект собран в ходе обучения автоматизации тестирования на Python 3, Selenium + Python.

backend

    mapsAPI - Google map API
        метод POST - создание локации. Ответ содержит уникальный place_id.
        https://rahulshettyacademy.com/maps/api/place/add/json?key=qaclick123 + тело запроса
        метод DELETE - удаление локации.
        https://rahulshettyacademy.com/maps/api/place/delete/json?key=qaclick123 + тело запроса

        Задача:
        Отправить 5 запросов на создание новой локации на карте.
        Сохранить уникальные place_id локаций в текстовый файл.
        Отправить запросы на удаление второй и четвертой локации, используя данные из текстового файла.


    SWAPI - API с информацией о Звездных Войнах.
        метод GET - получение информации о персонаже. Ответ содержит объект с эндпоинтами на все фильмы, в которых
        принимал участие персонаж.
        https://swapi.dev/api/people/:id/

        метод GET - получение информации о фильме. Ответ содержит объект с эндпоинтами на всех персонажей, которые
        принимали участие в фильме.
        https://swapi.dev/api/films/:id/

        Задача:
        Отправить запрос на получение информации о персонаже Дарт Вейдер (id = 4).
        Получить имена всех персонажей, которые появлялись в одних фильмах с Дартом вейдером.
        Удалить все дубли имен персонажей, сохранить список уникальных имен в текстовый файл.


frontend

    demoqa - ресурс с элементами фронта для написания автотестов.
        https://demoqa.com/date-picker/

        Задача:
        Выполнение без использования локаторов конкретной даты на календаре.
        Ввести в поле Select Date дату, увеличенную на 10 дней по отношению к сегодняшней.


    saucedemo - прототип интернет-магазина.
    https://www.saucedemo.com/

        Задача:
        Провести тестирование сценария оформления заказа на 2 товара.
        В ходе оформления заказа сверять наименование и стоимость товаров до помещения в корзину, в корзине,
        на странице оформления заказа.
        Сверить расчет общей стоимости товаров.
